# 🎉 图表和字体修改完成说明

## ✅ 修改完成

### 1. 📊 **股票详情图表修改：成交量 → 涨跌幅**

#### 修改内容：
- **图表tooltip显示**: 从"成交量"改为"涨跌幅"
- **数据来源**: 使用`pct_chg`字段（涨跌幅百分比）
- **显示格式**: `涨跌幅: +1.23%` 或 `涨跌幅: -2.45%`

#### 具体修改：
- **后端API**: `app.py` 历史数据接口增加 `pct_chg` 字段
- **前端JS**: `stock_detail.js` 将 `volumes` 改为 `changePcts`
- **tooltip**: 显示格式化的涨跌幅而非成交量

#### 验证数据：
- 平安银行(000001)近期数据：
  - 2025-09-01: 涨跌幅 -1.33%
  - 2025-09-02: 涨跌幅 +0.76%
  - 2025-09-03: 涨跌幅 -1.92%

### 2. 🔤 **全站字体加粗**

#### 加粗范围：
- ✅ **页面正文**: `body` 全局字体加粗
- ✅ **英文数字**: `en-font` 类字体加粗
- ✅ **数字字体**: `number` 类字体加粗
- ✅ **表格标题**: `th` 元素字体加粗
- ✅ **表格内容**: `td` 元素字体加粗
- ✅ **输入控件**: `input, button, select, textarea` 字体加粗
- ✅ **图表字体**: Chart.js 图例、坐标轴标题、刻度标签加粗

#### CSS修改详情：
```css
/* 全局字体加粗 */
body { font-weight: bold; }

/* 英文和数字字体加粗 */
.en-font { font-weight: bold; }
.number { font-weight: bold; }

/* 表格字体加粗 */
.stock-table th { font-weight: bold; }
.stock-table td { font-weight: bold; }

/* 控件字体加粗 */
input, button, select, textarea { font-weight: bold; }
```

#### JavaScript图表字体加粗：
```javascript
// 图例、坐标轴标题、刻度标签都设置为 weight: 'bold'
font: {
    family: '"微软雅黑", "Microsoft YaHei", sans-serif',
    weight: 'bold'
}
```

## 🔧 技术细节

### 涨跌幅数据处理：
- **数据来源**: tushare API的 `pct_chg` 字段
- **格式化**: 正数前加"+"号，负数自带"-"号
- **精度**: 保留2位小数

### 字体加粗实现：
- **CSS层面**: 全局设置 `font-weight: bold`
- **JavaScript层面**: Chart.js配置中设置 `weight: 'bold'`
- **继承机制**: 子元素自动继承父元素的加粗设置
